# 🌊 DamBTI (ダムBTI)

**「あなたの心は何㎥ですか？」**

全3000基以上の日本全国のダムデータから、あなたの性格にもっとも近いダム（ソウルダム）を見つけ出す狂気の診断アプリです。

## ⚠️ 概要 (Overview)

これはMBTI（16タイプ診断）ではありません。**ダムです。**
重力式コンクリート、アーチ式、ロックフィル...。
あなたの放流スタイル、貯水容量、そして堤高を科学的（？）に分析し、実在するダムとマッチングさせます。

### 特徴
- **リアルタイム連携**: 特定のダム（徳山ダム等）とシンクロした場合、国交省データから**今の水位**を引っこ抜いてきます。雨が降ればあなたのダムも潤います。
- **狂気のUI**: コンクリートの重厚感と緊急放流の緊張感を演出。
- **シェア機能**: 診断結果を誇らしげにX（旧Twitter）に放流できます。

## 🚀 使い方 (How to Play)

1. **[診断サイト]** にアクセス（デプロイURL貼ってね！）
2. **「ゲートを開放する」** ボタンを連打（1回でいいです）
3. 18問の質問に、己の魂に従って答える。
4. **結果発表**。あなたの正体が判明します。推しダムに会いに行きましょう。

## 🛠 技術スタック (Tech Stack)

「無駄にモダン」な構成で作られています。

- **Frontend**: Astro + Svelte + Vite (爆速)
- **Backend**: Node.js + Express + TypeScript
- **Data**: Wikidata (SPARQL) + 国交省 川の防災情報 (Scraping)
- **Infra**: Railway (Backend) + Netlify (Frontend)

## 🤝 免責事項

- この診断はエンターテイメントです。結果が「黒部ダム」でも、明日から観光放流ができるわけではありません。
- リアルタイムデータは「取れるときだけ取る」というゆるい仕様です。取れなくても泣かない。
