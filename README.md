# ダムBTIつくった

自分の性格がどのダムっぽいか診断するやつです。
なんかMBTI流行ってるからダムでやったら面白そうかなって思って作りました。

## やれること
質問に答えると似てるダムが出ます。2200基以上のダムデータがあります。
Wikiからデータ引っこ抜いてきて、堤高とか貯水量でマッチングしてます。

あと国交省のサイトからリアルタイムの水量とか取って来れるようにしたんですが、
全部のダムでやると大変なので徳山ダムとか有名どころだけ対応してます。
データ取れないときは「取れません」って出ます。仕様です。
雨降ったときとかに見ると面白いかも。

## 中身
無駄にAstroとかSvelteとか使ってます。
バックエンドはNode.jsです。
RailwayとNetlifyで動いてます。
最初シミュレーションで適当な値だそうとしたら怒られたのでちゃんとスクレイピングしてます。

あくまでネタなので、診断結果見て「へー」ってなってくれたら満足です。
不具合あったらごめん。

## 今後の予定
特にないけど、気が向いたら対応ダム増やすかも。
ダムカードとか連携できたらいいなー。
